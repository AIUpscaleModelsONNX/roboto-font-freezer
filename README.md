# Roboto Font Freezer

## Project Description

The Roboto Font Freezer is a project designed to optimize and streamline the usage of the Roboto font in applications. It allows users to package and utilize only the specific font weights and styles that they need, reducing the overall font file size and improving load times.

## Setup Instructions

1. **Clone the Repository**  
   First, clone the repository to your local machine:
   ```bash
   git clone https://github.com/AIUpscaleModelsONNX/roboto-font-freezer.git
   ```

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd roboto-font-freezer
   npm install
   ```

3. **Usage**  
   To freeze the desired font weights and styles, run the following command:
   ```bash
   npm run freeze
   ```

4. **Build**  
   After freezing, build the project:
   ```bash
   npm run build
   ```

This will create the necessary font files that can be used in your application.
